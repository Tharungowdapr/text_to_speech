# Design Context

## Brand / Personality
QuickTime/iTunes aesthetic — dark brushed metal with glassmorphism panels, Apple HIG-inspired. Clean, minimal, professional.

## Typography
- System font stack (-apple-system, BlinkMacSystemFont, "Segoe UI")
- 14px body, 11-12px for UI chrome (labels, counters)
- No custom/imported fonts (Inter, Roboto)

## Color Palette
- Background: #1c1c1e to #2c2c2e (brushed metal)
- Text: #f5f5f7 primary, rgba(255,255,255,0.55) secondary
- Accent: #0071e3 (Apple blue)
- Success: #30d158, Error: #ff453a, Warning: #ff9f0a
- Borders: rgba(255,255,255,0.10) subtle
- No gradients on text, no glowing accents, no purple/violet/cyan

## Layout
- App shell: sidebar (200px) + main content + bottom transport bar
- Sidebar: iTunes source-list style with section labels
- Transport bar: QuickTime-style at bottom (play/pause, skip, scrubber, volume, speed)
- Cards: glassmorphism with backdrop-filter blur

## Motion
- Only opacity and background-color transitions (0.12-0.15s ease)
- No bounce/elastic easing
- No layout property animations (width, height, padding, margin)
- Use transform: scaleX for progress bar
